# Water-level-indicator.pcb

📖 Project Overview

This project is a simple and useful Water Level Indicator designed using a custom PCB. The main purpose of this system is to monitor the water level inside a tank and provide indications when the water reaches different levels. It helps avoid water overflow and reduces unnecessary water wastage. In this project, I designed the complete PCB layout for the circuit, which made the system more compact, reliable, and easier to use compared to a breadboard setup. The PCB includes sensor input connections, indicator LEDs, transistor stages, resistors, and power supply connections arranged neatly for smooth operation.

⚙️ Working Principle

The circuit works by using probes placed at different heights inside the water tank. When water touches a probe, it completes the circuit and activates the corresponding transistor, which turns ON the LED indicator. As the water level increases, more LEDs glow to indicate the level of water present in the tank. A buzzer can also be activated at the maximum level to alert the user.

🔧 PCB Design Process

The PCB was designed by first creating the circuit schematic in PCB design software such as KiCad or EasyEDA. After assigning footprints to all components, the components were arranged neatly to minimize track crossing and improve routing. Thicker tracks were used for power connections, and proper spacing was maintained between components for easy soldering and testing. The final layout was checked using DRC/ERC before generating Gerber files.
